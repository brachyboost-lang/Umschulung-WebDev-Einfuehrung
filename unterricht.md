Umgang mit Visual Studio Code kurz durchgesprochen

Inline suggestions aktiviert. Das geht ganz einfach in dem man in die Einstellungen geht (Strg + ,) und dort nach "Inline Suggestions" sucht. Dort kann man die Funktion aktivieren oder deaktivieren.

Lehrer erklärt wie man Extensions installiert. Dazu geht man in den Extensions Tab (das Symbol mit den vier Quadraten auf der linken Seite) und sucht nach der gewünschten Extension. Dann klickt man auf "Installieren". Wir haben uns heute die Extensions GitHub Theme als beispiel angeschaut und installiert.
