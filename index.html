Inline Suggestions aktiviert. Das geht in den Einstellungen (Strg + ,) unter "Inline Suggestions"; dort lässt sich die Funktion ein- oder ausschalten.

Erster HTML-Code

Mit <!DOCTYPE html> wird der Browser angewiesen, die Seite als HTML5 zu interpretieren.

Mit <html lang="de"> wird die Sprache der Seite auf Deutsch gesetzt. Das ist wichtig für Suchmaschinen und Screenreader.

Mit <head> werden Metadaten der Seite definiert, wie z. B. Titel und Zeichencodierung.

Mit <meta charset="UTF-8"> wird die Zeichencodierung auf UTF-8 gesetzt, was die meisten Zeichen unterstützt.

Mit <meta name="viewport" content="width=device-width, initial-scale=1.0"> wird die Seite für mobile Geräte optimiert.

Mit <title> wird der Titel der Seite festgelegt, der im Browser-Tab angezeigt wird.

Der sichtbare Hauptinhalt gehört in das <body>-Element.

Zum Fettsetzen von Text verwendet man <b>fetter Text</b> oder semantisch besser <strong>wichtiger Text</strong>. Die Schreibweise </b> allein ist falsch (fehlendes öffnendes Tag).

Extensions

Extensions installiert man über den Extensions-Tab (Symbol links). Suche die gewünschte Extension und klicke auf "Installieren". Beispiel: "GitHub Theme".

HTML-Tags

Überschriften:

  • <h1> für die Hauptüberschrift
  • <h2> für Unterüberschriften
  • <h3> für weitere Unterüberschriften
  • <h4> bis <h6> für noch tiefere Ebenen

Listen:

  • <ul> für ungeordnete Listen (Punkte)
  • <ol> für geordnete Listen (Nummern)
  • <li> für Listenelemente

Absätze und Hervorhebungen:

  • <p> für Absätze
  • <b> und <strong> für fett
  • <i> und <em> für kursiv

Struktur-Tags:

  • <article> für eigenständige Inhalte
  • <nav> für Navigationslinks
  • <section> für thematische Abschnitte
  • <header> für Kopfbereiche
  • <footer> für Fußzeilen
  • Home
  • About
  • Contact

Links Beispiele:

  • <a href="https://example.com">Linktext</a> — so erstellt man einen Hyperlink.

Sonstiges:

  • <br> für Zeilenumbrüche (sparsam verwenden)
  • <pre> für vorformatierte Texte
  • <blockquote> für längere Zitate

Formulare

Ein korrektes Formular-Beispiel:

<form action="/submit" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <input type="submit" value="Absenden">
</form>

Allgemeine Hinweise:

  • action gibt die URL an, an die das Formular gesendet wird (z. B. "/submit").
  • method ist üblicherweise "get" oder "post".
  • <label for="id"> verbindet das Label mit einem Eingabefeld via dessen id.
  • <fieldset> und <legend> gruppieren Formularfelder.
