# WEB-Entwicklung

Web-Entwicklung bezieht sich auf die Erstellung und Wartung von Websites und Webanwendungen. Sie umfasst verschiedene Aspekte wie Webdesign, Webprogrammierung, Datenbankverwaltung und Serverkonfiguration.

Es umfasst unteranderem die erstellung von Webseiten mit HTML, CSS und JavaScript, die Entwicklung von Backend-Logik mit Programmiersprachen wie Python, Ruby oder PHP sowie die Nutzung von Frameworks und Bibliotheken zur Vereinfachung des Entwicklungsprozesses.

Was unser Webbrowser anzeigt, wird durch HTML (HyperText Markup Language) strukturiert, CSS (Cascading Style Sheets) gestaltet und durch JavaScript interaktiv gemacht.
